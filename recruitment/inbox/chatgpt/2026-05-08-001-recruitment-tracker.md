## Boundaries

This file extracts only recruitment tracker facts from prior ChatGPT Portfolio + Recruitment project context.

Excluded:
- Full company research
- Cover letters and recruiter message drafts
- Interview prep content
- Compensation strategy
- Private emotional reflections
- Detailed portfolio case study content
- AutoScout24 internal metrics or sensitive claims

Portfolio relevance is kept high level only, such as:
- AI design leadership
- Marketplace/search systems
- Cross-platform product design
- Automotive marketplace experience

## Status legend

Allowed statuses:
- Lead
- Researching
- Preparing
- Applied
- Recruiter screen
- Interviewing
- Assignment
- Final stage
- Offer
- Rejected
- Withdrawn
- Archived

Priority labels are inferred from prior chats only where there was enough signal:
- P0 = highest active priority
- P1 = strong target
- P2 = possible target / useful signal
- P3 = low priority / weaker fit / mostly exploratory
- Unknown = not enough context
