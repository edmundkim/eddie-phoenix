# ChatGPT Import Inbox

Use this folder to bring over context from previous ChatGPT job-search work.

## Recommended Flow

1. Put raw pasted exports or long transcripts in `raw/`. That folder is ignored by git.
2. Create a short distilled note in this folder when the raw context contains durable value.
3. Move structured facts into:
   - `applications.md` for pipeline status.
   - `companies/` for company or role research.
   - `materials.md` for reusable message or portfolio material context.
4. Delete or archive raw files when they are no longer needed.

## Distillation Checklist

- What company, role, or application does this context affect?
- Is the information a fact, a draft, an opinion, or a decision?
- Is there a date or source attached?
- Should it become tracker data, company research, material guidance, or private context?
- Does anything contain sensitive information that should stay in `private/`?
