# Company Notes

Create one Markdown file per serious company or opportunity. Use `templates/company-research.md` as the starting point.

Suggested filename pattern:

```text
company-name.md
company-name-role-title.md
```

Keep notes concise and sourced. A useful company note should make three things clear:

1. Why this company might be worth Eddie's attention.
2. What role or problem Eddie could credibly help with.
3. What next action would move the opportunity forward.
